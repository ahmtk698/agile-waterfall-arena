# Deploy Guide — Agile vs Waterfall Arena

## Quick Start (No Build Required)

This project is a single self-contained HTML file. No build step, no server, no dependencies.

### Option 1 — Open Locally
1. Download `src/index.html`
2. Open it in any modern browser (Chrome, Firefox, Edge, Safari)
3. The game runs immediately

### Option 2 — GitHub Pages (Live URL)
1. Push `src/index.html` to a public GitHub repository
2. Go to **Settings → Pages → Branch: main → Save**
3. Wait 1–2 minutes
4. Access at: `https://<username>.github.io/<repo-name>`

### Option 3 — Any Static Host
Upload `src/index.html` to any static hosting service (Netlify, Vercel, etc.) — no configuration needed.

## Notes
- No npm install, no build command, no environment variables
- No internet connection required after the file is loaded
- File size: ~83 KB
