## 📖 About

A browser-based gamification project for **COM0463 IT Project Management**.

Players take the role of a Project Manager at a global tech firm and must choose between **Agile** and **Waterfall** methodologies across 10 real-world scenarios of increasing complexity — managing crises, budgets, team morale, and stakeholder pressure along the way.

---

## 🗂️ Game Levels

| Level | Title | Scenarios | Timer | Highlights |
|-------|-------|-----------|-------|------------|
| 1 | Briefing Room | 2 | None | Tutorial mode, detailed guidance |
| 2 | Methodology Wars | 3 | 60s | Stakeholder conflicts, Iron Triangle |
| 3 | Crisis Management | 3 | 45s | Random crises, dashboard bars |
| 4 | Executive Sim | 2 | 40s | Programme-scale, earn your PM title |

---

## 🎯 Learning Objectives (Bloom's Taxonomy)

| Level | Outcome |
|-------|---------|
| L3 Apply | Select the correct methodology based on project conditions |
| L4 Analyse | Analyse requirement stability, risk, and stakeholder complexity |
| L5 Evaluate | Evaluate trade-offs and justify methodology decisions |
| L6 Create | Create optimal project strategies under real-world constraints |

## 🏆 Scoring

- ✅ Correct methodology selection → **+100 pts**
- ✅ Each correct decision → **+60 pts**
- ⚡ Speed bonus (under 12s) → **+25 pts**
- 🔥 Crisis resolved → **+160 pts**
- ❌ Wrong answer penalty (Level 2+) → **−25 to −50 pts**

**PM Titles:** Junior Analyst → Project Manager → Senior PM → Executive PM 👔
